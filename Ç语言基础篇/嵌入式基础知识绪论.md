**四、**<font size=5 face="黑体"> 计算机数据表示 </font>

- 将送入计算机的各种信息转换成二进制（0和1）的数据后，计算机才可以处理并运算。

- 计算机常见的计数有二进制、八进制、十进制、十六进制。

- 能够进行算术运算得到明确数值概念的信息成为计算机数据数据，其余的为非数值数据。

- 对于符号类数据则采用查询八位二进制来存储，对于汉字采用16位二进制数来表示。

- <font size=4 face=黑体>二进制、十六进制</font>
  
  - 二进制由0和1组成，逢2进1。即二进制1111为十进制15。
  
  - 十六进制由&emsp;0，1,…,10,A,…,F&emsp;组成，逢16进位，可以和二进制较快的转换。
  
    <span style="font-family:'Wingdings 2'">R</span>例如：0X1010=1x16<sup>3</sup>+0x16<sup>2</sup>+1x16<sup>1</sup>+0x16<sup>0</sup>=4096+16=5012
  
- <font size=4 face=黑体>非数值数据表示</font>

  <span style="font-family:'Wingdings 2'">R</span>计算机中数据的概念是广义的，除前面提到的数据类型外还有文字、符号、图像、逻辑信息等等，都已以0、1形式存在，所以称为非数值数据。

  <span style="font-family:'Wingdings 2'">R</span>字符数据主要是指数字、字母、通用符号等等，在机器内都转换成计算机能识别的二进制编码形式，普遍采用ASCII码。

  <span style="font-family:'Wingdings 2'">R</span>第一部分由 00H 到 1FH 共 32 个，一般用来通讯或作为控制之用，有些字符可显示于屏幕，有些则无法显示在屏幕上，但能看到其效果(例如换行字符、归位字符)。

  <span style="font-family:'Wingdings 2'">R</span>第二部分是由 20H 到 7FH 共 96 个，这 96 个字符是用来表示阿拉伯数字、英文字母大小写和底线、括号等符号，都可以显示在屏幕上。

  <span style="font-family:'Wingdings 2'">R</span>第三部分由 80H 到 0FFH 共 128 个字符，一般称为【扩充字符】，这 128 个扩充字符是由 IBM 制定的，并非标准的 ASCII码。这些字符是用来表示框线、音标和其它欧洲非英语系的字母。

- 语法符号

    <span style="font-family:'Wingdings 2'">R</span> 任何高级语言都有自定义的词法符号和支持的数据类型。
    <span style="font-family:'Wingdings 2'">R</span> 词法符号是语言的基本组成单位，数据类型是数据的基本属性。
    <span style="font-family:'Wingdings 2'">R</span> 词法符号是程序设计语言中有若干字符组成的有意义的最小语法单位。
    <span style="font-family:'Wingdings 2'">R</span>  按照词法符号在程序中的作用，可以分为：关键字、 标识符、分隔符、运算符和标点符号
    <span style="font-family:'Wingdings 2'">R</span> 关键字是由系统预定义的词法符号，有特定的含义，不允许用户重新定义。

    &emsp;&emsp;1) 标识符由一个或多个字母、数字或下划线组成
    &emsp;&emsp;2）标识符的第一个字符必须是字母或下划线

    <span style="font-family:'Wingdings 2'">R</span>运算符是表示运算的词法符号，C语言有非常丰富的运算符，按功能可以分为：

    <span style="font-family:'Wingdings 2'">R</span>标点符号，C语言中的标点符号有逗号、分号、冒号、花括号、圆括号。标点符号的作用与分隔符相似，但用法非常严格，有着明确的语法规定。有些标点符号出现在表达式中时，当作运算符使用。例如：


- 调试

<span style="font-family:'Wingdings 2'">R</span>在C程序调试时，可以利用`__FILE__,__FUNCTION__,__LINE__`,实现代码跟踪调试

```c
//若这个C代码文件名为ex.c
#include<stdio.h>
int main(int argc,const char *argv[])
{
    printf(" %S,%s,%d \n",__FILE__,__FUNCTION__,__LINE__);
    return 0;
}

//运行结果为 ex.c(源程序文件名),main(使用了什么函数),5(行数)
```
